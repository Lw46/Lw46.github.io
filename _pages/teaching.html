Data description

An atmospheric correction process was performed for Band01 to 06 of the CEReS Grid AHI using the look-up table (LUT) method based on the Second Simulation of a Satellite Signal in the Solar Spectrum vector (6SV) radiative transfer model (RTM) and the Copernicus Atmosphere Monitoring Service Reanalysis (CAMSRA) - European Centre for Medium-Range Weather Forecasts (ECMWF) Atmospheric Composition Reanalysis 4 (EAC4).

Cloud mask data were generated using the Yamamoto et al. (2018, 2022) algorithm, which sets thresholds for reflectance and brightness temperature in visible and infrared band data to identify clouds. The algorithm provides a cloud mask of the FD region with a spatial resolution of 0.02°.
